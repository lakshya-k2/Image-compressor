A simple web tool built with Next.js to quickly optimize, resize, and compress your images.

Features:

Batch Processing: Upload and optimize multiple images at once.
Smart Compression: Intelligently handles PNGs and JPEGs to ensure the best compression, reducing file size without forcing format changes.
Resize Presets: Easily resize images for Instagram, Twitter, or keep the original dimensions.
Adjustable Quality: Use a slider to control the final quality and compression level.
Download All: Get all your optimized images in a single ZIP file.
Instant Stats: See exactly how much space you saved in KB and as a percentage.

Tech Stack
Frontend: Next.js, React, Tailwind CSS
Backend: Next.js API Routes, sharp for image processing

First, run the development server:

```bash
npm install
npm run dev
```
